An Android Application to manage a collection of physical objects.

  • Source code

This app captures the details of an object of your choice in their collection and displays the information in a visually appealing manner. the idea behind the application is to keep track of items you might lend out to friends in order to not forget who has what item. items such as books, tools, or accessories

Stack used for this project:

  • Programming Languages:
    • Java
    • Kotlin
  • Development Environment:
    • Android Studio: The official IDE for Android development.
  • UI/UX Design Tools:
    • XML Layouts: Used to create the user interface.
  • Backend Services:
    • Firebase: Provides authentication, real-time database, cloud storage.
  • Version Control:
    • Git
